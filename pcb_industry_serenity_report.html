一、行业全景：为什么PCB是AI的"血管"

什么是PCB？

PCB（印制电路板）是所有电子设备的"神经系统基座"——芯片再强，没有PCB就是一块废硅。每块GPU、每台服务器、每个交换机的信号传输，全靠PCB上的铜线完成。

为什么AI时代爆发？

一句话：单台AI服务器的PCB价值量是传统服务器的10倍。

传统服务器用6-8层普通板就够了，AI服务器需要28-78层高速高频板，工艺难度逼近封装基板级别。英伟达GB200/GB300单机需要的PCB面积和层数是传统的5-10倍，而且必须用超低损耗材料。这不是量的增长，是质的飞跃。

二、产业链瓶颈定位（核心）

PCB产业链结构：上游材料 → 中游覆铜板(CCL) → 下游PCB板厂 → 终端AI服务器

Serenity方法论：沿产业链向上游追溯，找到"一旦断货，万亿产业就停工"的刚性瓶颈。

瓶颈层级图谱

S级 电子布（玻纤布）— 第一卡脖子

为什么是S级？

  • 供给刚性极端：高端织布机交付排期到2030年，设备交付周期6年。新增产能无法快速释放。
  • 涨价验证：2026年内已五轮涨价，7628电子布较年初涨超70%，高端布价格翻倍
  • 垄断格局：日本日东纺垄断全球约90%超薄低介电玻纤布产能，主动控制扩产节奏
  • 锁单现象：谷歌、Meta、英伟达等已签2-3年长约锁定全部日东纺产能，国内厂拿货排队12-18个月
  • 缺货率：>60%，是整条链最紧缺的环节

国内标的：宏和科技(603256)、中国巨石(600176)、国际复材(301526)

判断：国内仅能小批量生产二代超薄布，与日系龙头存在明显技术代差。但正因为此，一旦突破，估值弹性巨大。宏和科技是电子布最纯标的。

A级 高频PPE/BT特种树脂 — 第二卡脖子

为什么是A级？

  • 绝对垄断：BT树脂由日本三菱瓦斯化学独家把控；PPE核心原料由沙特SABIC垄断
  • 交期极长：高端BT树脂交期拉长至9-15个月，现货有价无市
  • 成本占比最高：树脂占覆铜板生产成本26-27%，是决定CCL等级的"灵魂材料"
  • 国内唯一量产：圣泉集团PPO产线满产，月订单560吨 vs 实际供应仅150吨，供需比4:1
  • 涨价猛烈：高端PPO树脂从35-40万/吨涨至60-80万/吨

核心标的：圣泉集团(605589) — 国内唯一PPO量产企业

判断：树脂壁垒最高但国产化最难。圣泉产能缺口巨大但扩产需时间，确定性强但爆发节奏取决于产能释放。

B+级 化学法球形硅微粉 — 隐藏的预期差

为什么特别标注？

  • 最大预期差：是整条链唯一"供需缺口逐月扩大但价格尚未明显上涨"的环节
  • 扩产最慢：化学合成产线依赖进口设备，建设+认证需2年以上
  • 下游倒逼涨价：电子布、树脂都涨了，硅微粉迟早被传导
  • 估值相对合理：不像电子布那样已被市场充分定价

核心标的：联瑞新材(688300)

判断：价格未启动=潜在爆发力最大。但需要确认涨价信号落地再动手。

B级 超低轮廓铜箔(HVLP)

  • 高端四代铜箔仍依赖日系，国内头部已实现二三代量产
  • 缺口约40%，但铜箔扩产周期相对较短（1-2年），供给刚性弱于电子布和树脂
  • 标的：铜冠铜箔(301217)、诺德股份(600110)、德福科技(301511)
  • 问题：铜冠铜箔PE 1009倍，诺德/德福今日跌停——市场已充分定价甚至过度炒作

中游 覆铜板(CCL) — 受益但非真瓶颈

  • 生益科技(600183)：龙头中的龙头，2025营收284亿+39%，但市值4078亿、PE 104，弹性有限
  • 南亚新材(688519)：高频高速CCL纯度最高，年涨389%，但PE 253估值极端
  • 判断：CCL厂受益于涨价传导（上游涨→CCL涨→转嫁下游），但自身不是供给瓶颈，产能可以扩

下游 PCB板厂 — 利润被上游吃掉的"打工仔"

  • 深南电路、沪电股份、鹏鼎控股：营收高增长，但原材料涨价挤压毛利率
  • 产能扩张590亿→高端产能仍紧缺，但长期壁垒不如上游
  • 鹏鼎大股东高位套现35亿，这是危险信号
  • 判断：板厂赚"辛苦钱"，真正的超额利润在上游材料。除非有独特技术壁垒（如IC载板），否则长期不选下游。

为什么跌？

  1. 资金跷跷板效应：英伟达Rubin全液冷官宣 → 液冷板块集体涨停 → 资金从PCB抽血到液冷
  2. 获利了结：PCB板块年内普涨100-400%，获利盘巨大，任何风吹草动都会触发踩踏
  3. 涨价预期"利好出尽"：建滔第五次涨价、木林森再提10%——涨价消息已持续数周，边际刺激递减
  4. 大股东套现信号：鹏鼎控股大股东高位精准套现35亿，机构嗅到风险

Serenity判断：健康回调，不是趋势反转

核心逻辑未变：覆铜板涨价传导顺畅、织布机6年排产无法改变、AI订单持续超预期——这些结构性利好不会因为一天的资金跷跷板而消失。

但短期确实过热：科翔+408%、南亚+389%、铜冠PE 1009——这种估值水平在任何回调中都会被放大。今天的跌幅是前期暴涨的代价。

结论：趋势不变，但需要消化估值。预计还有1-2周震荡整理，等7月中报预告确认业绩后再看。

五、投资节奏判断

现在能买吗？

今天不接飞刀，但也不要空等"完美回调"。

"等回调"是分析师思维的通病——追求完美入场点。但实际投资中，完美入场点只存在于回测里。正确做法是用仓位管理替代择时判断。

观察建底仓本周(6/23-27)今天PCB大跌本身就是低吸窗口15%挑最确定的1-2只试水，每只≤总投入5%

仓位纪律

  • 单只个股仓位不超过总投入的20%，分散3-4只
  • 止损线：单只从建仓成本跌-15%强制减半；跌-25%清仓
  • 止盈线：翻倍后减1/3锁定利润，剩余跟踪产业节奏
  • PCB赛道建议配置占组合60%（产业链更厚、弹性标的更多）

核心心法

"左侧试仓靠胆量，右侧加仓靠证据。"

今天PCB大跌 → 可以小仓位低吸（赌的是产业趋势不变）
中报确认 → 加仓（有证据了）
Q3旺季 → 攻击（产业逻辑+业绩+情绪三共振）

今天就是个活生生的例子：PCB大跌+液冷暴涨——如果要同时布局两个赛道，今天PCB低吸建底仓，液冷等分时回调再动手，天然形成了分批节奏。

七、最大风险

⚠️ 风险排序（从高到低）

1. 估值泡沫崩塌（最大风险）

PCB板块年涨100-400%，PE动辄100-1000倍。一旦某家公司中报业绩不及预期，30-50%杀估值不是危言耸听。尤其是铜冠铜箔PE 1009、兴森PE 613这类标的，任何负面消息都是灭顶之灾。

应对：只买PE<100的真瓶颈标的，远离PE>300的纯炒作。

2. AI资本开支不及预期

整条PCB链的逻辑建立在"AI服务器持续高增长"上。如果2026下半年云厂商资本开支放缓，或英伟达Rubin量产延期，高端PCB需求将出现环比回落，"110%增速"的预期会被修正。

应对：密切跟踪英伟达/微软/Meta每季财报中的capex指引。

3. 涨价传导断裂

上游材料涨价→CCL涨价→PCB涨价→终端客户接受。如果终端客户（英伟达、字节等）拒绝接受涨价或转向海外替代供应商，涨价链条会断裂。目前看传导顺畅，但不可掉以轻心。

4. 国产替代不及预期

宏和科技、圣泉集团的投资逻辑包含"国产替代"溢价。但如果高端产品良率/性能长期不达标，客户不敢切换供应商，那国产替代就只是故事。

应对：关注客户认证进展公告。通过认证=逻辑兑现；认证失败=估值杀。

5. 资金面风格切换

今天就是活生生的例子——液冷爆发，PCB被抽血。如果市场风格从"科技成长"切向"低估值红利"，整个赛道股都会遭殃。
